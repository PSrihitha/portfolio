Project Title: Online Auction System(IconicBid)

Short Project Overview:

IconicBid is a futuristic online auction platform built using the MERN stack (MongoDB, Express, React, Node.js). It offers a sleek, glassmorphic UI for a premium user experience. Users can securely register, create auctions, and place real-time bids. The platform is styled with a black and silver theme, using Bootstrap and custom CSS. Designed for elegance and performance, IconicBid brings a cinematic feel to digital bidding.

Project Details

    Tech Stack:

  • Frontend: React (with Vite), Bootstrap 5, Custom CSS, React Router
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: JWT (JSON Web Token)
  • Other Features: Axios, CORS, Glassmorphism UI, Particle Background

Project Title: Job Portal frontend using React Vite

Short Project Overview:

JobConnect is a responsive and performance-optimized job portal frontend application built using React with Vite. It provides a seamless user interface for job seekers to find and apply for jobs, and for employers to post and manage job listings. This frontend project focuses purely on the UI and UX, with static data handling in preparation for future backend integration.
